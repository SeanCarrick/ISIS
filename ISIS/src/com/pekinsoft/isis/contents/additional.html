Additional JAR Files

On this step, you may select additional JAR files that your project may require. If they are all in the same folder, you can select all of them at once, contiguously, by selecting the first file, holding down the <Shift> key while clicking on the last file. If you need to select non-contiguous files, hold down the <Ctrl> key while clicking on each file that you need.

If you need JAR files from multiple folders, you may add each one individually by clicking on the "Select Additional Jar Files" button to select each additional file.

Library Files

If your project requires additional libraries, such as the Apache Derby JAR for database access, You will select them on the next step.
