Select the Project JAR File

On this step, you need to select your project's JAR file that provides the execution of your application. If your project encompasses multiple JAR files, you may only select the primary JAR file on this step. If you have other JAR files you need to include, you will be able to include them on the next step.

Additional JAR Files

Once you have selected your project's primary JAR file, click on the Next button. On the next step, you will be able to select additional JAR files, if needed. Select your additional JAR files, or just click Next, if none are needed.

Library Files

If your project requires additional libraries, such as the Apache Derby JAR for database access, You will select them on the next step.
