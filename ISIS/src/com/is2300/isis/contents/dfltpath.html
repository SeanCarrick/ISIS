Default Installation Path

On this step, you need to enter the default installation location for your application. Typically, on Windows™, this would be in either the "Program Files" (for 64-bit software) or, "Program Files(86)" (for 32-bit software). On Linux, this can either be in the "/usr/" or the ""/usr/local/" folder. However, the user can also choose to have the default install location be in their home folder: "C:\Users\UserName\" on Windows™; or "/home/userName/" on Linux systems.

You may select from the list of environment variables by selecting the checkbox, then choosing which environment variable to use for your default install location. If a location meeting your needs does not appear in this list, simply uncheck the checkbox and manually enter the path into which you wan your software installed.
